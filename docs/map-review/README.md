# Map Review Artifacts

This directory now serves as archival design/reference material for the older
map-rendering workflow.

## Current Status

- The shipped `generate-map` CLI is packet-only. It validates a section,
  builds layout/topology data, and writes `<id>-packet.md`.
- The snapshot/style audit commands referenced in earlier iterations are not
  part of the current `package.json` surface.
- Files in this folder are kept for research, design notes, and local
  experimentation only; they are not part of the active verification pipeline.

For the commands that are currently supported, use the root
[`README.md`](../../README.md).
